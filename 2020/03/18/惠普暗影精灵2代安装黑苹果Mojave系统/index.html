惠普暗影精灵2代安装黑苹果Mojave系统

惠普暗影精灵2代安装黑苹果Mojave系统

一、下载必要软件

1、下载Paragon Hard Disk Manager // 处理磁盘，U盘 这个软件比较大

2、下载 TransMac // 制作macos启动盘

3、下载os系统 下载 macOS-Mojave系统，网上很多

这里提供百度网盘下载链接：

链接：https://pan.baidu.com/s/1Ma1kGeUh8DLhGnlvf8yPEQ
提取码：m0ht

4、下载Clover Configurator 网上很多 #注意：如果系统镜像里有了clover就不用下载这个，我上面提供的就包含了，不用下载这个，所以下面也不会介绍这个的使用。

有必要说一下，由于这个系统自带的efi和我的电脑是适配的所以不用单独去找efi了，如果不适配需要在网上找到适配自己电脑的efi，不然无法引导进入你的os系统

二、在windows下的准备

1、制作启动盘

1）打开 Transmac

2) 鼠标放在优盘标志上右击

3) 选择restore with Disk Image然后点击Yes

4) 然后在重复一下 2） 3)的操作,会弹出一个文件选择框，选择你下载的os镜像，然后两次都点yes

之后就是漫长的等待，很长时间。你现在就可以去做其他的事情了

5） 完成之后会弹出，该优盘无法识别，需要格式化，记住一定不要格式化，点击取消就可以，到此为止你的os启动盘就做好了。

2、准备安装系统的磁盘，我自己安装的是mojave版本的系统，这个必须安装在电脑的主磁盘上。

使用Paragon Hard Disk Manager

首先划分出一定大小的空间，然后删除该分区，在创建分区，

然后点击 More options

选择文件系统分区格式为Apple HFS 然后点击Create

点击右上角的Apply 使你刚才的操作生效

到此为止你的准备操作就做好了。

然后就是设置一下bios,关闭 安全模式 ，从u盘启动。

三、开始安装

1、刚开始都是按照指导来安装，

date 042310152015.30 //在正式安装之前要在终端输入这条指令
